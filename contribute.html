Why?

PHPedia is open! So, we need your help.

  • We need you to create posts
  • We need you to comment on post and issues
  • We need you to open new issues
  • We need you to contribute!

How?

All you need to do is:

  • Fork us on Github
  • Create a post, or contribute to an existing one
  • Send it to us opening a pull request

We currently support Markdown and Textile

Approval policy

New post:

  • Create a new post and send it to us
  • PHPedia team will check the post and accept it, if valid

Contributing to an existing post:

  • Fork it
  • Contribute
  • Open a pull request
  • PHPedia team will open an issue to notify the last commiter of the post
  • PHPedia team and the last commiter will validate the contribution
  • If the last commiter does not answer the issues within one week, PHPedia team will validate your contribution and approve it, if valid

We just want to remind you, that at any time you can discuss why or why not your contribution or post should be approved.
